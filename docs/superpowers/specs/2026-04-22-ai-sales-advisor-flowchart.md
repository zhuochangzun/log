## 4. SKILL 技能映射

| SKILL ID | 名称 | 触发场景 |
|----------|------|----------|
| SKL101 | 接待咨询 | 客户来访、意图识别 |
| SKL102 | 智能报价 | 询价、费用计算 |
| SKL103 | 下单创建 | 确认下单、创建订单 |
| SKL104 | 收款确认 | 账单生成、付款确认 |
| SKL105 | 跟单状态 | 物流跟踪、状态更新 |
| SKL106 | 催付提醒 | 未付款订单提醒 |
| SKL107 | 主动营销 | 外呼、销售机会跟进 |
| SKL108 | 产品推送 | 新产品、促销活动 |

---

## 5. 关键节点

| 节点 | 说明 | 异常处理 |
|------|------|----------|
| 出库前校验 | 必须确认账单已付款 | 未付款拦截，触发催付 |
| AI无法识别 | 意图不明确、复杂问题 | 转人工助手 |
| 收款状态 | 轮询/回调确认 | 超时冻结订单 |
| 人工介入 | 客户要求电话、投诉等 | AI保留上下文 |
