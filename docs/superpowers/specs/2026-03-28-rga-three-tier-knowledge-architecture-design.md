# RGA 三层知识体系架构设计

**日期**: 2026-03-28
**状态**: 已批准
**版本**: v1.1

---

## 1. 背景

وسع RGA 知识平台服务于睿云公司，面向国际快递行业，提供 AI 中台能力。

**三层用户**：
1. **睿云平台** — 管理行业基础知识
2. **货代公司** — 管理客户公司数据
3. **货代员工/客户** — 使用系统的人员

## 2. 三层知识架构

```
平台层（全局共享）
    │
    ▼
公司层（租户隔离）
    │
    ├── 订单层（订单数据）
    │
    └── 用户层（个人数据）
```

### 2.1 层级权限

| 层级 | 可见范围 |
|------|---------|
| 平台层 | 全部可见（所有公司、所有员工用户） |
| 公司层 | 仅该公司可见（员工 + 用户） |
| 订单层 | 仅该公司可见，按订单阶段隔离 |
| 用户层 | 仅本人可见，支持公司授权共享 |

## 3. 知识分类体系

### 3.1 平台层分类

| 分类 | 说明 |
|------|------|
| 快递公司 | FedEx/DHL/UPS/顺丰等 |
| 电商平台 | Amazon/eBay/Shopify等 |
| 海关政策 | 各国家进出口规定 |
| 物流方案 | FBA头程、海外仓、直邮等 |
| 行业术语 | CIF/FOB/ENS/BL等 |
| 竞品动态 | 行业新闻、竞品分析 |

### 3.2 公司层分类

| 分类 | 说明 |
|------|------|
| 公司资料 | 资质、协议、合作条款 |
| 产品信息 | 航线、时效、价格表 |
| 客户信息 | 客户档案、联系人 |
| SOP流程 | 内部操作标准流程 |
| 报价规则 | 折扣政策、账期规则 |
| 常见问题 | 客户高频问题回复 |

### 3.3 订单层分类

| 分类 | 说明 |
|------|------|
| 待处理 | 新下单未确认 |
| 执行中 | 已确认处理中 |
| 已完成 | 正常完结 |
| 异常 | 问题件、退件等 |

### 3.4 用户层分类

| 分类 | 说明 | 共享 |
|------|------|------|
| 个人话术 | 常用回复模板 | 可授权 |
| 常用模板 | 邮件/消息模板 | 可授权 |
| 学习笔记 | 个人学习记录 | 不可 |
| 收藏内容 | 收藏的知识条目 | 不可 |

## 4. SKILL 与知识层映射

| SKILL | 知识层 | 数据来源 |
|-------|--------|---------|
| 运单追踪 | 公司层 + 订单层 | TMS Tool |
| 报价 | 平台层 + 公司层 | TMS Tool + 知识库 |
| 库存查询 | 公司层 | WMS Tool |
| 创建订单 | 公司层 + 订单层 | TMS/OMS Tool |
| 知识检索 | 三层全部 | RAG 知识库 |
| 客户信息 | 公司层 + 用户层 | 公司层 + 授权 |

## 5. 检索流程

```
用户 query
    │
    ▼
SKILL 匹配
    │
    ▼
知识分类路由 → 确定候选层
    │
    ▼
语义补充判断 → 扩展检索层
    │
    ▼
权限过滤 → 按角色过滤
    │
    ▼
合并排序 → 返回结果
```

### 5.1 兜底策略

```
用户层 → 公司层 → 平台层
```
